# PaletteStudio 🎨

<img src="/assets/images/logo.webp" height="100" width="100">

Generate sophisticated and harmonious color palettes for your product designs with PaletteStudio - a simple yet powerful color palette generator.

## ✨ Features

- **Unique Palette Generation**: Create beautiful color combinations using our custom algorithm
- **One-Click Copy**: Simply click on any color to copy its HEX code
- **Interactive UI**: Hover over colors to reveal HEX codes and lock buttons
- **Color Locking**: Lock specific colors while generating new palettes
- **Share Functionality**: Easily share your favorite color combinations
- **Clean Interface**: Colors displayed in simple, easy-to-view divs

## 🛠️ Installation

1. Clone the repository:
```bash
git clone https://github.com/iambhvsh/PaletteStudio.git
```

2. Navigate to the project directory:
```bash
cd PaletteStudio
```

3. Open `index.html` in your preferred browser

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Clipboard API

## 📖 Usage

1. **Generate a Palette**
   - Click the "Generate" button to create a new color palette
   - Each generation creates a unique set of harmonious colors

2. **Copy Colors**
   - Click on any color to automatically copy its HEX code to clipboard
   - Hover over colors to view their HEX codes

3. **Lock Colors**
   - Hover over a color to reveal the lock button
   - Lock specific colors to keep them while generating new palettes

4. **Share Palettes**
   - Use the share button to share your favorite color combinations
